# spider_on_gamersky
爬取gamersky上的图片

# 之前的想法
> 先获取网站上所有的网址信息
> 然后在对网址中的图片进行多线程下载
## 问题
> 当程序报错时,没有添加"重连"方法
> 对下载的网址,图片也没有详细的log

# 目前的想法
> * 为了保证速度,肯定要使用多线程的
>   * 那么问题是应该如何选择呢?
>   * **获取每个网址内的所有图片地址**
>   * **然后将这些地址放入到多线程中进行下载**
>   * 回去先看看书,确定如何多线程下载
> * 对于下载的网址,图片的log进行记录
> * 我要保证需要下载的东西必须下载完毕且不会重复下载
> * 写好每一个类,每一个方法
> * 学会单元测试
> * 代码符合pylint规范(虽然里面很多规则很麻烦)

# 步骤
> 1. 获取所有标题的网址
> 1. 获取标题内的图片网址
> 1. 根据图片网址内容,下载图片
